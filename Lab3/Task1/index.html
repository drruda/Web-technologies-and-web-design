Практичну роботу виконала cтудентка групи ІТ-31/1 Руденко Дарина Євгенівна
Застосування каскадних таблиць стилів (CSS) для форматування HTML-документів. Робота з блоками.
Стилем або CSS (Cascading Style Sheets, каскадні таблиці стилів) називається набір параметрів форматування, що застосовується до елементів документа, щоб змінити їх зовнішній вигляд.
